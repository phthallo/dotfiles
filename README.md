Not finished by any means. Remind me to add setup scripts and to credi people whose work I've shamelessly adapted.

## Stuff
- WM: Hyprland
- Status bar: Waybar
- Dock: nwg-dock-hyprland
- Terminal emulator: Kitty 
- Terminal prompt: Starship
- File manager: Dolphin
- Screenshot: Flameshot
- Shell: zsh
- Launcher: Vicinae
- Browser: Firefox
- Logout: wlogout
- Spicetify[^1]
- Fastfetch 
- Symlink management: stow 

Config files go in `config/.config` (`~/dotfiles/config/.config`).

## Keybinds
- Super + Space for launcher 
- Super + B for browser
- Super + Enter for terminal
- Super + M for Spotify
- Super + Q to close the current program
- Super + Printscreen to screenshot
- Super + F for file manager

[^1]: Make sure to install Spotify using Flatpak.
